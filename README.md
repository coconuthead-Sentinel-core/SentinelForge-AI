## What this is

A public, full-stack AI workstation and orchestration platform built solo by a 26-year healthcare professional in career transition to AI systems engineering. SentinelForge AI implements a three-tier persistent memory model, MCP-server protocol layer, FastAPI backend, React frontend, and audit-trail discipline informed by clinical privacy-first instincts.

## Architecture

- **`01_BACK_END/`** — FastAPI service with query interface, document store, zone migration, and handoff persistence
- **`02_MIDDLE_LAYER/`** — Protocol activation, query routing, and pipeline definitions
- **`03_FRONT_END/`** — React component plus dashboard spec
- **`04_CONTEXT/`** — Glyphic Codex master, schemas, zone instructions, and project identity

## Three-Zone Cognitive Routing

- **GREEN** — load 7-10, active session work
- **YELLOW** — load 4-6, reference and synthesis
- **RED** — load 1-3, archived and crystallized

This zone system maps directly to Multi-Tiered System of Support (MTSS) clinical accommodation models and Asthma Action Plan urgency tiers — clinical frameworks formally validated through the architect's healthcare coursework.

## Quick Start

```bash
pip install fastapi "uvicorn[standard]" pydantic
cd 01_BACK_END
uvicorn server:app --reload --port 8000
```

Open <http://localhost:8000/docs> for the OpenAPI / Swagger UI.

## Tech Stack

- **Backend:** Python 3.11+, FastAPI, Pydantic v2
- **Storage (current):** Local JSON files for mock-first DX
- **Storage (planned):** Azure Cosmos DB
- **AI Services (planned):** Azure OpenAI
- **Protocol Layer (planned):** FastMCP
- **Frontend:** React JSX plus HTML/CSS/JS dashboard
